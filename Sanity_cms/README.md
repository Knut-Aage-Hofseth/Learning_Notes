# Hva er Sanity?
Sanity er ett *Content Management System* ofte forkortet til CMS. Dette betyr at Sanity organiserer innhold, som tekst og bilder som så presenteres ut til verden som nettsider.  
Sanity kan tilpasses og kodes om i, både for å lage verktøy i Sanity for redaktørene og for å lage komponenter som bestemmer oppførsel og utsende ute på nettsiden.
